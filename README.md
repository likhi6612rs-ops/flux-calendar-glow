# 🌟 flux-calendar-glow

An AI-driven productivity companion designed to transform chaotic student schedules into a high-performing lifestyle.

---

## 🚀 Project Overview
As students preparing for competitive entrance examinations, we face intense academic pressure and constant procrastination. I built **flux-calendar-glow** to act as a "digital nervous system." The goal is to personalize the student experience—balancing rigorous study milestones with a calming interface that relieves stress rather than creating it.

## 💡 Key Features
* **Integrated Academic Tracking:** Dedicated portal to map out examination dates, track syllabus chapters, and manage mock test schedules seamlessly.
* **Structured Study Timer:** A distraction-free environment designed for deep work, helping you maintain focus during intense study sessions.
* **Consistency Dashboard:** A specialized tracking system to monitor daily habits and mitigate procrastination, ensuring steady progress toward your goals.
* **Stress-Relieving Aesthetic:** Designed with a calming, lavender-toned interface to reduce cognitive load and prevent digital fatigue during long study hours.

## 🛠️ Architecture & Tech Stack
* **Frontend Framework:** React / TypeScript
* **Styling:** Tailwind CSS (featuring a custom, stress-relieving color palette)
* **Development Workflow:** Built using modern agile practices with **Lovable** as an advanced prototyping engine, allowing me to focus on structural systems thinking, user flow, and feature iteration.

## 📈 My Engineering Journey & Future Scope
* **The Challenge:** Designing an interface that displays complex scheduling data (exams, chapters, habits) without overwhelming the user.
* **The Solution:** A clean, modular layout coupled with a soothing color scheme keeps the interface feeling peaceful and approachable.
* **Future Scope:** I am currently developing a "Wellness & Vitality" pillar to track hydration, nutrition, and fitness, which will integrate into the dashboard to support students holistically.
